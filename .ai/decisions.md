# Decision Log

## Repository
- Repository is a knowledge platform, not a note collection.
- Countries are filters, not the primary abstraction.
- Research Ecosystems are first-class entities.
- Architecture is frozen.
- Quality Gates replace feature milestones.
- Repository Maintainer replaces Feature Builder mindset.
- Evidence is required for facts and relationships.
- Recommendation engine must be deterministic and explainable.
- Legacy content remains compatible during migration.
- AI context is stored in the repository, not chat history.
