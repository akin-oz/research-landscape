# Archived relationships

Move a relationship directory here only when it is intentionally inactive: the applicant has chosen another route, an application cycle has concluded, the advisor has declined contact, or an earlier record is retained for long-term reflection after admission.

## Archive rules

1. Keep evidence links and dated interaction facts intact.
2. Replace unresolved next actions with a closing note and an explicit archive reason.
3. Do not rewrite past impressions to make an outcome look inevitable.
4. Retain only material that remains appropriate, accurate, and lawful to store.
5. Preserve the advisor slug so scripts and historical links remain stable.

Archived does not mean failed. An archived record can document a useful conversation, a future collaboration lead, or a decision that improved the applicant’s research direction.

Never archive sensitive/private correspondence merely to keep it in version control. Remove it instead.
