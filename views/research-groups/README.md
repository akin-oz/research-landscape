# Research groups view

This public view answers: which reviewed Research Group records have valid,
evidence-backed direct hosts and documented research, software, and area paths?
Its canonical contract is `research-groups` in
[`../definitions.yaml`](../definitions.yaml).

The future generator must enforce ADR 0006 before including a group. It must
show only canonical links and concise metadata; it must not copy group analysis,
people lists, project lists, funding claims, or opportunity statements.
