# Waiting list view

`waiting-list` is a private overlay for canonical entities that an applicant
intends to revisit at a stated date. Its canonical contract is `waiting-list`
in [`../definitions.yaml`](../definitions.yaml).

It requires a private deferred entry and revisit date. Deferral is not a public
fact about an entity and cannot affect global, country, university, software,
ecosystem, or recommendation results. No public generated output exists.
