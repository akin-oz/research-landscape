# Principal investigators view

This public view answers: which reviewed Principal Investigator records have
documented affiliations, group leadership, research areas, or software links?
Its canonical membership, facets, ordering, and evidence policy are declared in
[`../definitions.yaml`](../definitions.yaml) as `principal-investigators`.

It displays concise metadata and canonical links only. A missing relation is
unknown, not evidence that a PI lacks a group, topic, software connection,
opportunity, or mentorship practice. Generated results are intentionally absent
until Quality Gate 6 provides deterministic generation and validation.
