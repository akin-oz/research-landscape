# Conferences view

This public view answers: which reviewed conference/event records connect to a
research area, organization, university, or ecosystem? Its canonical contract
is `conferences` in [`../definitions.yaml`](../definitions.yaml).

Event dates are part of membership and ordering. An event connection never
proves current attendance, a speaking opportunity, travel support, mentoring,
or a future event. Results are deliberately absent until deterministic
generation is implemented in Quality Gate 6.
