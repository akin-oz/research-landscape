# Current focus view

`current-focus` is the private, time-bounded successor to the compatibility
guide in [`current-targets`](../current-targets/README.md). Its canonical
contract is `current-focus` in [`../definitions.yaml`](../definitions.yaml).

Membership requires a private shortlist entry with an unexpired next action.
It never writes applicant plans, contact history, scores, or subjective
priorities back to a canonical entity. No public generated output exists.
