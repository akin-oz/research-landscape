# Open source

The most direct external route is an AiiDA contribution: improve a plugin, test an edge case, clarify a provenance/tutorial example or reproduce a workflow bug. Pair it with a small Quantum ESPRESSO/DFT study so the contribution demonstrates scientific judgement.

Evidence: [AiiDA documentation](https://aiida.readthedocs.io/) and [team page](https://www.aiida.net/sections/team.html).
