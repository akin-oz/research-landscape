# Software culture

OQMD is an openly accessible DFT database with a REST/OPTIMADE API and CC-BY 4.0 data statement ([source](https://www.oqmd.org/)). The group states it develops and maintains OQMD ([source](https://www.wolverton.northwestern.edu/)).

This is strong evidence of a data/platform culture. It is insufficient to claim a particular programming language, test framework or internal development process without inspecting current public code.
