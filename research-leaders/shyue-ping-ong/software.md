# Software culture

`pymatgen` is direct evidence of Python-oriented materials software. The Materials Project catalogue calls it Python code for materials genomics and lists it alongside workflow and API tooling ([source](https://perssongroup.lbl.gov/software.html)); its project documentation is public ([source](https://pymatgen.org/)).

Evidence supports open source and Scientific Python. It does not alone establish the lab’s present CI/review policy; verify the repository’s current workflows and contribution guide.
