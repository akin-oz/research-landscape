# Software culture

## Direct evidence

The group’s software page says Materials Project code is hosted in its GitHub organization and lists `pymatgen`, FireWorks, Jobflow, Custodian, atomate/atomate2, CrystalToolkit, Emmet, `mp-api` and Matbench ([source](https://perssongroup.lbl.gov/software.html)). It also states an intention to publish code promptly with new research.

## Assessment

Open source, Python, workflow automation and public documentation are **high-confidence** signals. The page does not by itself prove the use of particular testing, CI, packaging or code-review policies in every subproject; inspect the current repositories before making those claims.
