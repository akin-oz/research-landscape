# Open-source and community surface

The official catalogue is the evidence: the programme publishes a family of public libraries, workflow tools, data models and API clients ([source](https://perssongroup.lbl.gov/software.html)). Materials Project also provides a public data/discovery platform ([source](https://materialssciences.lbl.gov/research/research-programs/materials-project/)).

## Practical entry route

Read contribution guides and current issue trackers for `pymatgen`, `atomate2`, Jobflow or `mp-api`; reproduce a small bug or improve an example before proposing architecture. Never assume a maintainer has capacity for unsolicited large features.
