# Software culture

The Materials Project software catalogue names public Python/data/workflow tools—`pymatgen`, FireWorks, Jobflow, Custodian, atomate2, Emmet and `mp-api`—and states that code is hosted in its GitHub organisation ([source](https://perssongroup.lbl.gov/software.html)). Jain’s profile documents leadership within the programme ([source](https://eta.lbl.gov/people/anubhav-jain)).

**Evidence boundary:** programme involvement supports a high scientific-software fit, but not a claim that every individual project has a particular CI, review or packaging policy.
