# Contact strategy

**Best timing:** after a Materials Project-compatible artifact or, for a public deadline, immediately with an honest work-in-progress link.

**Recommended message:** state the exact current research direction you read; link one small reproducible study or reviewed upstream contribution; ask whether a narrowly framed workflow/data/ML question is relevant to a listed project. Use a normal professional email for research/admissions; use GitHub only for the software contribution itself.
