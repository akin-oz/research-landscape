# Open source

Materials Project provides a meaningful contribution surface: public code, data APIs, workflow tooling and benchmarks ([source](https://perssongroup.lbl.gov/software.html)). The strongest portfolio link is a small, reviewed contribution or reproducible application of that stack, not a generic claim of AI interest.

Possible entry points: documentation tests, data-model examples, benchmark reproduction, API client edge cases or workflow provenance improvements—after reading each project’s contribution guidance.
