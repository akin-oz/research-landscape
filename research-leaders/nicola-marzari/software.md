# Software culture

AiiDA is described as a joint effort of PSI’s Materials Software and Data group, EPFL THEOS and Bosch RTC, with workflows, provenance, HPC interfaces, open science and open source as explicit concepts ([source](https://www.aiida.net/sections/team.html)). Materials Cloud provides open computational-science sharing services ([source](https://www.materialscloud.org/)).

This is high-confidence evidence of ecosystem software culture; it should not be misread as authorship/maintenance attribution for every project.
