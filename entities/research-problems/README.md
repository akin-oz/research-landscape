# Research Problems

This directory is the canonical home for evidence-bounded Research Problem
records. The reviewed records are [Lattice Thermal Conductivity Prediction](lattice-thermal-conductivity-prediction.md),
[Crystal Symmetry Determination](crystal-symmetry-determination.md), and
[Machine-Learned Interatomic Potential Modeling](machine-learned-interatomic-potential-modeling.md),
and [Density-Functional Electronic-Structure Calculation](density-functional-electronic-structure-calculation.md).
[Materials Property Prediction](materials-property-prediction.md) is also a reviewed record.
A problem is a named computational challenge, not a project, funding call,
benchmark, or ranking of scientific importance.
