# Research Ecosystems

This directory is the canonical home for Research Ecosystem records, connecting people, labs, software, institutions, funding, communities, and conferences. The reviewed records are [AiiDA Ecosystem](aiida.md), [ASE Ecosystem](ase.md), [AFLOW](aflow.md), [CP2K Ecosystem](cp2k.md), [FAIR Chemistry](fair-chem.md), [FAIRmat](fairmat.md), [LAMMPS Ecosystem](lammps.md), [Materials Cloud](materials-cloud.md), [Materials Project](materials-project.md), [MatML](matml.md), [Open Catalyst Project](open-catalyst-project.md), [OpenKIM Ecosystem](openkim.md), [OQMD](oqmd.md), and [Quantum ESPRESSO Ecosystem](quantum-espresso.md); Materials Cloud has an evidence-bounded historical connection to the NCCR MARVEL funding programme, and FAIRmat has an evidence-bounded connection to the [Eighth FAIRmat Users Meeting](../conferences/fairmat-users-meeting-2026.md). No existing ecosystem content was moved to create these records.
