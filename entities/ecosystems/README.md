# Research Ecosystems

This directory is the canonical home for Research Ecosystem records, connecting people, labs, software, institutions, funding, communities, and conferences. The reviewed records are [AiiDA Ecosystem](aiida.md), [Materials Project](materials-project.md), [OQMD](oqmd.md), [AFLOW](aflow.md), [Materials Cloud](materials-cloud.md), [FAIRmat](fairmat.md), and [ASE Ecosystem](ase.md); Materials Cloud has an evidence-bounded historical connection to the NCCR MARVEL funding programme. No existing ecosystem content was moved to create these records.
