# Countries

This directory is the canonical home for Country records. Countries are filters and relationship endpoints in the entity model, not the repository's central abstraction; the reviewed records are [Denmark](denmark.md), [Germany](germany.md), [Italy](italy.md), [Japan](japan.md), [Singapore](singapore.md), [Switzerland](switzerland.md), [United Kingdom](united-kingdom.md), and [United States](united-states.md), created without moving existing country content.
