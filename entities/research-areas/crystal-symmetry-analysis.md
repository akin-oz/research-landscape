# Crystal Symmetry Analysis

Crystal Symmetry Analysis is a controlled research area for entities whose own
reviewed evidence explicitly describes finding, handling, or analyzing crystal
symmetries. It provides a topic-first discovery path without treating every
crystal-structure workflow or materials project as symmetry analysis.

## Evidence

| Source ID | Evidence |
| --- | --- |
| `SRC-SPGLIB-DOCUMENTATION` | [Spglib documentation](https://spglib.readthedocs.io/en/stable/) describes Spglib as a library for finding and handling crystal symmetries, with functions for symmetry operations, space-group identification, Wyckoff assignment, structure refinement, primitive-cell search, and irreducible k-points. Accessed 2026-07-13. |

## Boundary and limitations

The label is used only when an entity's own evidence explicitly describes
crystal-symmetry analysis. It does not rank research problems, methods,
software, groups, advisors, or institutions, or classify every crystallography,
materials-informatics, electronic-structure, or atomistic-simulation activity.
