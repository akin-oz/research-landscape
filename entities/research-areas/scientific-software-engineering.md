# Scientific Software Engineering

Scientific Software Engineering is a controlled research-area label for work
explicitly concerned with developing, engineering, maintaining, or extending
software for scientific computation. It is deliberately narrower than simply
using research software and distinct from the scientific domain that software
serves.

## Boundary and limitations

This controlled label is used only where a record's own sources directly
describe scientific-software development, engineering, maintenance, or
infrastructure contribution. A public repository, use of a code, data analysis,
high-performance computing, or Computational Materials Science alone does not
establish this classification. It is not a quality, employability, mentorship,
or software-maturity score.
