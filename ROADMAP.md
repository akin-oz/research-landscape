# Roadmap

## v0.1 — Infrastructure

- Establish templates, governance, methodology, metric contracts, and validation placeholders.

## v0.2 — Method validation

- Validate definitions with contributor review; publish machine-readable schema and Markdown linting.

## v0.3 — Turkey pilot

- Publish a small, consent- and evidence-reviewed pilot with coverage and limitations clearly stated.

## v0.4 — Automation

- Add reproducible source ingestion, citation checks, score generation, and freshness monitoring.

## v0.5 — Europe expansion

- Extend the validated workflow to additional countries with regional-source guidance.

## v1.0 — Public release

- Release stable schemas, reviewed reports, documentation, and governance process.

## Future

- Global coverage, multilingual documentation, API/data exports, and community-maintained source adapters.
