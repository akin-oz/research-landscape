# Scoring models

Each directory is an immutable major scoring model. Reports reference the directory and document revision they used, for example `v1/advisor.md`. Changing a formula, normalization rule, default weight, or interpretation requires a new major directory; clarifications that do not change semantics are recorded in the file history.

`v1` is a design contract only. It does not authorize scoring until metric definitions, evidence requirements, and validation tooling are complete.
