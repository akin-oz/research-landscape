# Ecosystems

The repository evaluates connected research ecosystems rather than treating countries as a proxy for opportunity. The current comparative assessment is [Global research ecosystems](../reports/global-ecosystems.md); its evidence sources are in the [global source register](../reports/global-sources.md).

New ecosystem records should identify the actual connection: shared software, data infrastructure, research facilities, funding programme, collaboration network or documented mobility route. They must not infer culture, visas, work-life balance or mentoring from geography.
