# Research Landscape v0.3.0 — Knowledge graph execution

This release completes the execution-phase Quality Gates for the current
reviewed canonical cohort.

## Highlights

- 67 reviewed v2 canonical entities and 116 evidence-bearing typed relationships.
- Evidence-bounded Software Ecosystem and Research Group Intelligence coverage.
- Thirteen deterministic view definitions, ten generated public views, and three deliberately private view contracts.
- Persistent validation, migration checks, health reporting, recommendations, regression tests, and CI drift checks.
- Contributor onboarding, entity authoring, review, community, automation, and recommendation guidance.

## Boundaries

v0.3.0 is not a prestige ranking, global census, mentorship ranking, or
admissions service. It does not infer language, opportunities, funding,
mentorship, or personal feasibility from missing evidence. Generated outputs are
projections; canonical entities remain the sole owner of reusable public facts.
