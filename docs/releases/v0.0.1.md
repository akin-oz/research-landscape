# Research Landscape v0.0.1 — Repository Foundation

This is the first public release of Research Landscape.

Rather than publishing data immediately, this release establishes the project's architecture, methodology, and engineering standards.

## Highlights

- Initial repository architecture
- Project documentation
- Research methodology
- Scoring philosophy
- Contribution guidelines
- Roadmap
- Open-source infrastructure

## What is intentionally missing?

This release does **not** include university analyses, advisor rankings, department reports, or scoring results. These will be added only after the underlying data model and evidence framework are finalized.

## Philosophy

Research Landscape does not rank researchers by prestige. Its goal is to help applicants identify research environments that best match their interests, experience, and long-term goals using transparent, evidence-based methods.

## Next milestone

The next release will introduce the repository's core knowledge model: entity definitions, a relationship model, metadata specification, stable identifiers, an evidence model, and JSON schemas. This foundation will support scalable analysis of universities, departments, research groups, and advisors.
