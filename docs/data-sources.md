# Data sources

Preferred source order: official entity pages and records; funder/publisher/repository records; persistent scholarly identifiers and indexes; reputable secondary sources. Record source owner, a public URL, date published if known, date accessed, relevant excerpt/value, and limitations. In a canonical entity's `## Evidence` table, the validator requires one unique `SRC-*` row per source, a public URL, and an ISO access date written as `Accessed YYYY-MM-DD`. Respect terms of service, robots controls, copyright, privacy, and rate limits. Do not collect non-public or sensitive personal data.
