# Migration status

**Status date:** 2026-07-12

The incremental migration pattern is proven and active for the current
canonical cohort. Legacy location-oriented reports, dossiers, and country
material remain in place as compatibility and historical-analysis surfaces.
Reusable public facts introduced during execution are owned by v2 records under
`entities/`; views, reports, recommendations, and generated outputs link to
those records rather than replace legacy material in bulk.

The validator enforces that v2 records live in their approved entity directories
and reports 67 such canonical records in the current cohort. This is not a
claim that every legacy document is migrated. Future work remains incremental:
create and evidence a canonical owner, validate it, add compatibility links,
then regenerate derived outputs.
