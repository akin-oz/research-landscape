# Global country context

Country pages are context, not rankings. The present global work is organized by connected research ecosystems in [the ecosystems report](../../reports/global-ecosystems.md), because funding routes, programme language, immigration rules, research software culture and supervision vary substantially within a country.

Before a country-specific application recommendation is made, collect dated primary evidence for the exact programme, funding scheme, work authorization route, language requirement and supervisor eligibility. Link that evidence to a specific ecosystem and lab.
